# Contributing

Thanks for contributing to `awesome-munich-data`.

This repository is intended to be a curated list of open data sources, APIs, tools, applications, and related resources for Munich.

## What To Add

Add resources that are clearly relevant to Munich and fit one of the existing sections in `README.md`, for example:

- open datasets
- APIs or API documentation
- civic tech tools
- interactive maps or visualizations
- data journalism or research projects

## Entry Guidelines

Each entry should:

- link to the primary project, dataset, or official source
- include a short description explaining what it is
- stay concise and factual
- mention important caveats such as unofficial status or outdated data when relevant

## Before Opening A Pull Request

Please check that:

- the resource is actually about Munich
- it fits the most appropriate section
- it is not already listed
- the description matches the surrounding style
- the link works

## Updating Existing Entries

Improvements are welcome, especially:

- fixing broken links
- clarifying descriptions
- removing duplicates
- marking outdated resources
- improving source quality

## Pull Requests

Small, focused pull requests are preferred.
